# Проектная работа «Закрывающий тег»

Одностраничный сайт с интерактивными карточками, анимациями и адаптивным дизайном. Проект помогает рефлексировать пройденный путь обучения и выразить личные эмоции через контент.

## 📸 Скриншот проекта

![Интерфейс сайта «Закрывающий тег»](/images/screen-4.png)

*Главная страница: сетка карточек с изображениями и текстом, интерактивные кнопки (Like, «Сохранить на память»), всплывающее окно с иконкой дискеты.*

## 🎯 Цель проекта

Отработать навыки:
* вёрстки с вариативными шрифтами (Inter) и акцентными шрифтами (PressStart2P);
* создания сложных фоновых эффектов (множественный градиент);
* работы с SVG‑символами и анимацией отдельных частей SVG;
* реализации интерактивных элементов (кнопки с эффектами при наведении и фокусе);
* адаптивной вёрстки по принципу Mobile First;
* использования CSS‑переменных и режимов смешивания (`mix-blend-mode`);
* создания анимаций через `transition` и `@keyframes`.

## ✅ Реализованные функции

- **Статичная версия** с подключением шрифтов, настройкой фона и разметкой всех блоков.
- **Адаптивность** под разные разрешения (включая контрольную точку 375 px).
- **Сложный фон** страницы: множественный градиент из трёх слоёв (сетка из полосок, вертикальный градиент, плавный переход цвета сверху вниз).
- **Интерактивные карточки** с лейблами (текст с обводкой и режимом смешивания `mix-blend-mode: hard-light`).
- **Всплывающее окно** (`<dialog>`) с открытием по кнопке «Сохранить на память» и закрытием по кнопке «ОК».
- **Фильтры на изображениях** карточек (минимум семь разных фильтров, один — множественный).
- **Анимации кнопок** при фокусе и наведении (тень, псевдоэлемент с эффектом движения, изменение цвета текста через `mix-blend-mode`).
- **Анимация иконки сердца** в трёх состояниях:
  * `:hover` — последовательное заполнение цветом сердцевины и основного тела;
  * `:active` — заполнение красным цветом сердцевины и тела (контур остаётся прежним);
  * клик (класс `.is-liked`) — заполнение всех частей, масштабирование сердца, вспышка искр.

## 🛠️ Использованные технологии

- **HTML5** — семантическая разметка, использование `<dialog>`, инлайн‑SVG для анимации сердца, мнемоники `&lt;` и `&gt;` для символов `<` и `>`.
- **CSS3**:
  * вариативные шрифты (`font-variation-settings`) и подключение через `@font-face`;
  * CSS‑переменные (`--accent-color`, `--animation-fill-color` и др.) в `variables.css`;
  * множественный фон через `repeating-linear-gradient` и `linear-gradient`;
  * `clamp()` для адаптивных размеров элементов;
  * режимы смешивания (`mix-blend-mode`) и обводка текста через `@supports`;
  * фильтры (`filter`) для изображений карточек;
  * анимации через `transition` (сердце) и `@keyframes` (масштабирование, искры);
  * логические свойства и гибкие отступы.
- **SVG** — оптимизация кода, группировка элементов (`<g>`), классы для анимации (`core`, `contour`, `main-body`, `sparks`, `heart`).
- **JavaScript** — скрипт для переключения состояния лайка (добавление/удаление класса `.is-liked`).

## 📝 Примечания

- Вёрстка выполнена по макету с соблюдением Pixel Perfect.
- Стили организованы в отдельных CSS‑файлах: переменные (`variables.css`), базовые стили (`style.css`), анимации (`animations.css`).
- Подключены фавиконки разных форматов (`.ico`, `.svg`, `apple-touch-icon`).
- Для фона страницы использована CSS‑переменная с множественным градиентом.
- Иконка сердца анимируется через изменение `fill` и `transform`, с задержками и плавностью.
- Всплывающее окно стилизовано через псевдоэлемент `::backdrop`.
- Все интерактивные элементы имеют состояния `:hover`, `:focus-visible`, `:active`.
- Проверена корректность отображения на разных разрешениях и в разных темах.
